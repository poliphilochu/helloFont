單位分成「絕對單位」和「相對單位」，其中又以 px、pt、em、% 這四種單位較為常見，其他還有 in、cm、ex 等等，相關介紹請參考 CSS Values and Units Module Level 3 的第 4 章與第 5 章。pt 是一個標準長度單位，適用於輸出，在螢幕顯示上因各家系統預設 dpi 不盡相同，這會造成同樣的 pt 值在不同的系統上顯示的大小不一致，故不適用於網頁設計。px 是螢幕上顯示的最小單位，可精準控制畫面的顯示效果，是設計師常用的單位之一。用於顯示在螢幕上的單位 W3C 建議使用 px、em 和 %。

單位分成「絕對單位」和「相對單位」，其中又以 px、pt、em、% 這四種單位較為常見，其他還有 in、cm、ex 等等，相關介紹請參考 CSS Values and Units Module Level 3 的第 4 章與第 5 章。pt 是一個標準長度單位，適用於輸出，在螢幕顯示上因各家系統預設 dpi 不盡相同，這會造成同樣的 pt 值在不同的系統上顯示的大小不一致，故不適用於網頁設計。px 是螢幕上顯示的最小單位，可精準控制畫面的顯示效果，是設計師常用的單位之一。用於顯示在螢幕上的單位 W3C 建議使用 px、em 和 %。
